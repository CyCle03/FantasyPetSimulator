# SD Fantasy Pet MVP Frontend

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Config
- Backend base URL uses `NEXT_PUBLIC_API_BASE` (default `http://localhost:8000`).
- Use `NEXT_PUBLIC_USE_PART_ASSETS=true` to render PNG parts.
- Use `NEXT_PUBLIC_ENABLE_MARKET=true` to show the market UI.
